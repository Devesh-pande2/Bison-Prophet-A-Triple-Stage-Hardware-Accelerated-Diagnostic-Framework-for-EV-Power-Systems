# Bison-Prophet-A-Triple-Stage-Hardware-Accelerated-Diagnostic-Framework-for-EV-Power-Systems
Bison-Prophet is an advanced Prognostic Health Management (PHM) ecosystem designed to predict high-voltage transients in EV charging infrastructure. This triple-stage framework integrates a Simulink plant simulator for high-fidelity pattern generation, an FPGA-ready diagnostic gateway for Verilog code generation and real-time threshold control, and a MATLAB-trained Bayesian Network.  By analyzing Low and Medium voltage patterns, the system performs Root Cause Analysis (RCA) to identify failing components—such as Capacitor Banks or MOSFETs—and provides a real-time "Spike Prophecy" before a 40V failure occurs. It bridges the gap between AI-driven software and high-speed hardware protection.
